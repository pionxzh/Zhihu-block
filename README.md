# Zhihu-Block

让知乎的黑名单更有用一些，自动屏蔽所有黑名单的文章/评论

# 使用方法

* 安装Tampermonkey(油猴)

  * 上官网 tampermonkey.net/ 下载
  * 或者在 Chrome应用商店里搜索`油猴` [下载扩展程序](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=zh-TW)

* 安装脚本
